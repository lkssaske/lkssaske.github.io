Installation Guide

Follow these steps in order to get FIFA 16 running correctly on your device:

  1. Download the APK file using the button below.
  2. Download the OBB file (it's a compressed .7z archive — extract it first).
  3. Install the APK, but don't open it yet.
  4. Move the extracted OBB folder to Android/obb/ on your device storage.
  5. Launch the game from your app drawer.
